## 🗂️ About This Repository

This is the source code for my **personal portfolio website** — a production-grade, modern web application that serves as both a professional showcase and a technical demonstration of what I can build.

> **Live at → [workwithfaran.com](https://workwithfaran.com)**

The portfolio itself is opinionated, fast, and built the way I'd build a real product — with proper architecture, type safety, CI/CD, and attention to UX details that most portfolio sites ignore.

---

## ✨ Portfolio Features

| Feature | Description |
|---|---|
| 🤖 **AI Chat Assistant** | Powered by Groq + Gemini — talks about my work, projects, and experience in real-time |
| 📬 **Dual-channel Contact Form** | Submissions hit my Telegram instantly *and* persist to Netlify Forms as a backup |
| 🏢 **Multi-position Career Timeline** | Visual promotion history per company — built with a custom data structure |
| 📝 **MDX Blog** | Fully rendered markdown blog with syntax highlighting |
| 🌗 **Dark / Light Mode** | System-aware, persisted theme with smooth transitions |
| 🎞️ **View Transitions API** | Page-to-page animations using `next-view-transitions` |
| 🖱️ **Smooth Scrolling** | Lenis-powered buttery scroll across all pages |
| 📊 **Analytics** | Privacy-first tracking via Umami (no cookies, GDPR compliant) |
| 🔒 **Rate Limiting** | IP-based rate limiting on the contact API to prevent spam |
| ⚡ **Edge-ready** | Deployed on Netlify with Node 22, optimized static output |

## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/Faranheit15/portfolio.git
cd portfolio

# Install dependencies (Bun preferred)
bun install

# Set up environment variables
cp .env.example .env.local
# Fill in: TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID, GEMINI_API_KEY,
#          GROQ_API_KEY, NEXT_PUBLIC_URL, NEXT_PUBLIC_UMAMI_SRC/ID

# Start dev server
bun dev
```

```bash
# Production build
bun run build
bun start
```

> Requires **Node.js 22+** or **Bun**
