# My Journey

From a curious CS student in Lucknow to a Fullstack & GenAI Engineer building production-grade tools for pharma and enterprise clients — here's how I got here.

## 2025

- **Promoted to Sr. SDE-1** at UsefulBI Corporation, leading fullstack and GenAI initiatives.
- Built **CSVT (Clinical Study Viability Tool)** — a Next.js 15 + FastAPI + PostgreSQL (AWS RDS) application with Azure AD authentication for clinical trial workflows.
- Developed **Meditwin** — a FastAPI + Supabase async backend for a patient digital twin concept, deployed on Railway, as part of an internal hackathon.
- Built **talent-ai** — an internal resume converter tool using Anthropic Claude via AWS Bedrock, React/shadcn frontend, and S3 storage, to assist the recruitment team.
- Created **Invoicey** — a Next.js invoice builder with Firebase Auth, MongoDB, and Gemini-powered natural language drafting.
- Mentored fullstack interns and conducted technical interviews for AI Full Stack Developer and GenAI Developer roles.
- Set up CI/CD pipelines (GitHub Actions → AWS EC2) and a fully customized WSL2 development environment.
- Launched redesigned personal portfolio with Journey and Blog sections.
- Active LinkedIn content creator covering AI, developer tooling, and industry trends.

## 2024

- Worked on **UBI Xcelerator** and **Protocol AI** at UsefulBI — building reusable React components and a React Flow Tree pipeline.
- Contributed to an **AI-powered medical report generation tool** for a pharmaceutical client, growing from frontend specialist to fullstack engineer with FastAPI.
- Led frontend development on a high-impact **Generative AI project** using React, Next.js, and FastAPI.
- Explored GenAI integrations, DevOps tooling (Docker, AWS), and modern runtimes like Bun.

## 2022–2023

- **Joined UsefulBI Corporation** as a Frontend Developer; quickly progressed to Software Engineer I and then Software Engineer II.
- Worked on **UBI PharmLaunch** and internal AI platforms in the pharma/clinical domain.
- Contributed to the internal **UBI Accelerator** platform — helped onboard new team members and drove frontend architecture decisions.
- Published blog posts on Medium about web development, MDX, and Next.js integration.

## 2021
